# lab-cost-dashboard


This project analyzes laboratory test costs and utilization across patients and CPT codes. The dashboard highlights key cost drivers and spending distribution.


Features
Total lab cost summary
Cost comparison by CPT code
Patient-level filtering
Percentage breakdown of lab spend
Identification of high-cost tests (e.g., CRP)
Tools Used
Microsoft Excel 
Pivot Tables / Data Modeling
Data Visualization

Dashboard Preview
Dataset
Contains sample patient lab test data
Fields include:
Patient ID
CPT Code
Lab Test Name
Cost

Key Insights
CRP accounts for ~67% of lab spending
Significant variation in cost by CPT code
A small number of tests drive most expenses

Future Improvements
Add time-series trends
Integrate real-world datasets
Build predictive cost model
